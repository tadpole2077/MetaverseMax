Plot Owner Report - User Guide [Release 0.5 Beta] (10/Apr/2021)

Production efficiency and history feature:

Click on the building type link on each row to open the history pop-up.

  • Lighting fast evaluation of all production run history since ownership of a selected Industry, Energy or Production building.
  • 2 types of efficienccy % evaluation : EFF(P), EFF(M)
  • EFF(P) - % based on total production starting zero
  • EFF(M) - % based on Min to Max, removing the base rate from the evaluation, each resouce has a different base, Wood having the larged base and easiest to produce
  • Use the P vs M comparision to identify the resouces where improved cits or applicances would have a bigger impact on production
  • 3 colour indicator of efficiency rating per run : red <50%, orange 50% to 80%, green>80%
  • Total runs and breakdown of total production per product since ownership
  • table pagination, sorting
  • Popup can be dragged and moved to optional position on page, new postion is reused on next use

Find the the building lifetime ROI, variance between different product efficiency with same cits and IP, optimal building upgrades and citizen assignment per product type, track how the successful players produce, impact from IP change ...

Stamina Alert feature:

Get the Max output from your buildings! Identify all your building containing low stamina citizens

Alert icon displays in right most column when minimum thresholds breached per building. Column is sortable.

On all buildings containing 1 or more citizens, icon of citizen with lowest stamina is shown, hover to find value

  • Background: When stamina falls below a certain threshold building production output reduces for that building.
  • Each Citizen within the building is evaluated, any below threshold will trigger the alert
  • Each building type has a custom minimum stamina threshold:
  • * Residential <=200
  • * Office <=50
  • * Commercial <= 10
  • * Municipal <=20
  • * Industrial <=50
  • * Production <=100
  • * Energy <=30
  • * Any other future new building type <=100 (default until specific threshold set in app)

  • Showing only filter building types matching buildings found within the current user portfolio
